## 2. General Mapping Principle

In ROMION:
- particles are **not** fundamental objects,
- they are **stable relational configurations**.

Mapping rule:

> A “particle” corresponds to a **class of loop states**
> with shared invariants and stabilization behavior.

No particle name implies a new entity.

## 3. Fermions as Loop States

### 3.1 Fermionic Character

Fermion‑like behavior arises from:
- loop stability,
- niche occupation,
- Topological Exclusion Principle.

Any loop state subject to exclusion
exhibits fermion‑like phenomenology.

---

### 3.2 Quarks

**Quarks** are mapped to:
- quark‑like Δ‑loops,
- admitting internal degeneracy,
- participating in composite bundles.

Key correspondences:
- color → internal degeneracy label,
- flavor → loop class and excitation level,
- mass → stabilization cost proxy μ.

Quarks do not exist independently;
they are meaningful only within composites.

---

### 3.3 Leptons

**Leptons** correspond to:
- isolated, non‑degenerate loop states,
- high individual stability,
- absence of binding requirements.

Examples:
- electron‑like states → stable single loops,
- neutrino‑like states → weakly interacting loop configurations.

---

## 4. Generations

Particle generations map to:
- excitation index n,
- loop metric length L₁,
- stability variation.

Higher generations:
- correspond to higher excitation,
- are structurally less stable,
- decay via permissible transformations.

Generational separation is structural, not fundamental.

---

## 5. Antiparticles

Antiparticles correspond to:
- conjugate loop states,
- reversed orientation σ,
- inverted topological charge Q_T.

Annihilation corresponds to:
- loop cancellation,
- release of relational modes,
- re‑distribution of relations.

## 6. Pauli Exclusion Reinterpreted

The Pauli Exclusion Principle is not imported.

It is **native to ROMION**
as the Topological Exclusion Principle:
no two identical loop states may occupy the same niche.

Particle‑physics exclusion behavior
is a direct projection of this structural rule.
